# scrcpy
Display and control your Android device.

```
kcp -u
kcp -i android-tools -d
kcp -i srccpy
```

Recommended for Android 12+, install [android-udev-rules](https://github.com/KaOS-Community-Packages/android-udev-rules)
```shell
kcp -u
kcp -i android-udev-rules -d
```
